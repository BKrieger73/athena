Welcome to the World of Python!

Once upon a time, knowing how to type on a computer and using Microsoft Word, Excel, and PowerPoint were very important skills to land high paying jobs. Nowadays, everyone knows how to type and use these basic programs. The skill you need to be successful today is programming.

What can you do with programming?

Today, almost everything electronic has some form of programming in it. A program is a set of instructions that performs a task that allows you to enhance the quality of life from convenience, e.g. allowing you to buy groceries online, to enjoyment, e.g. video games. Your smart phone and all the games you have installed on it are programs. Everything on the computer is a program. There are even high tech fridges that have programs to allow you change the temperature remotely or detect what food you need to buy the next time you go to the grocery store. Having the skill to write programs allow grants you the ability to enhance the quality of life in many ways from convenience (apps to turn off your lights remotely) to enjoyment (mobile and video games).

Why Python?

Python is a great language to start with because you can write programs that do big things with very little code. In addition, Python is a high level language, meaning it is closer to English than some other programming languages, making it easier to learn. However, the main difference between English as a language and a programming language is that in programming, your letters, symbols, and numbers (your code), must be perfect, e.g., there are no typos, extra characters, or symbols. When these errors occur, your program will not work.

How can I write my first program?

The following line is all you need to write your first program. Please type it into the scratch pad.

print "Hello, World!"

What this program does is print out "Hello, World" to your screen. Note that you must have double quotes at the beginning and end of your statement and you must type "print" without any typos. You can change the text inside the quotes to anything you want and it will change the text that is printed to the screen. That's it! Now you know how to write your first program - go ahead and try it in the first challenge.
