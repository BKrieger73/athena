In this lesson we will cover loops using Python!

A loop is a way of repeating the same block of code many times.

There are two main types of loops: For loops and While loops. In this lesson we will go over for loops.

This is what a for loop looks like in code:

for i in myArray:
     print(i)

In the above example, the print function will run for as many items as there are in the myArray variable.

Initialize an array of at least 2 elements and write a loop like the one above to print the contents of the array.
