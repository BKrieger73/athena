Welcome to the intro lesson for Python!

In this lesson you will learn how to use a function built into python called print.

What print does is writes a String (which for now can be thought of simply as an english sentence, although we will get into more depth later) to the console. The console is where we view the output of our little programs.

Follow the example to print the string "Hello World":

print("My first print")

Output: My first print

And remember! Python is case-sensitive! This means that print and Print are not the same thing!
