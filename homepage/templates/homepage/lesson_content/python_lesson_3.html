Lesson 3

Arithmetic

Math is everywhere. To calculate a 20% discount on an iPhone 7 that is $749, use the equation:
newPrice = 749 - 749 * 0.2

The resulting price is $599.20. In Python, the multiplication operator is *. In the example above, the result is stored in the variable, newPrice.

Variables can be used on the right hand side of an assignment. For example:
originalPrice = 749
newPrice = oldPrice - oldPrice * 0.2

The result is the same price, but in this case, we stored the original price of the iPhone in the variable, originalPrice and used it to compute the newPrice.

The same variable can be used on both sides of the assignment (equal sign). For example:
price = 749
price = price - price * 0.2

Again, the resulting price is $599.20. However, in this case, we used only one variable, price, to do the arithmetic. Note, the value of price was used for the computations on the right side before it was overwritten with the result.

In these three examples, note that the order of operations was followed. To review, operations are performed in the following order:
Parentheses, inside to outside
Exponents
Multiplication and Division, left to right
Addition and Subtraction, left to right
For example:
result = ((1 + 2)*3 + 3) ** 2 + 2 * 2 - 2 / 2

The result is 147. Note that the result is calculated as follows:
result = (3 * 3 + 3) ** 2 + 2 * 2 - 2 / 2
result = (9 + 3) ** 2 + 2 * 2 - 2 / 2
result = 12 ** 2 + 2 * 2 - 2 / 2
result = 144 + 2 * 2 - 2 / 2
result = 144 + 4 - 2 / 2
result = 144 + 4 - 1
result = 148 - 1
result = 147
